7 таблиць вже є в supabase-schema.sql. Нижче — що потрібно вписати вручну щоб сайт запрацював.

products
Мінімум 6–12 товарів. Обов'язкові поля:
slug — унікальна URL-назва (vyshyvanka-poltavska)
name_uk — назва українською
price — ціна у гривнях
category — vyshyvanky / sukni / rushnyky / dim / aksesuary
images — масив URL мінімум 2 фото
technique — Хрестик / Гладь / Мережка
description — опис 2–3 речення
is_new / is_bestseller — хоча б 2–3 з кожним true
Критично
profiles
Створюється автоматично при реєстрації (тригер в schema.sql).
Після першого входу:
UPDATE profiles SET role='admin' WHERE email='your@email.com';
Критично
posts
Мінімум 3–4 блог-статті:
slug, title, excerpt, content (markdown),
image (URL), author, category, read_time
Важливо
reviews
5–10 відгуків вручну щоб товари мали рейтинг:
product_id, author, rating (1–5),
text, verified=true
Тригер автоматично перерахує рейтинг у products
Важливо
orders
Заповнюється автоматично через checkout. Вручну нічого.
Авто
wishlist
Заповнюється коли юзер натискає ❤️.
Авто
addresses
Заповнюється з акаунту або checkout.
Авто
