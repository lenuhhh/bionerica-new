## 🎯 Использование компонентов

### CardStylePreset - Редактор дизайна

**Где используется:** Админ > Дизайн

```tsx
// Компонент self-contained, просто вставьте:
<CardStylePreset />

// Экспортирует JSON с текущим стилем
// Использует местное состояние для управления
```

**Функции:**
- ✅ 3 предустановки (Premium Dark, Minimal Light, Energetic Green)
- ✅ Редактирование 10 параметров стиля
- ✅ Живой предпросмотр с примером товара
- ✅ JSON экспорт для сохранения

### PushNotificationManager - Push-уведомления

**Где используется:** Админ > Настройки > Push-уведомлення

```tsx
// Использование с обработчиком отправки:
<PushNotificationManager
  onSend={async (notification) => {
    const response = await fetch('/api/admin/push', {
      method: 'POST',
      body: JSON.stringify(notification),
    })
    return response.ok
  }}
/>
```

**Функции:**
- ✅ Проверка разрешения браузера
- ✅ Форма для создания уведомления
- ✅ Тестирование локально
- ✅ История отправок
- ✅ Планирование (опционально)

## 🐛 Возможные проблемы и решения

### Проблема: Вкладка "Дизайн" не видна
**Решение:** Нажмите F5 для перезагрузки, очистите локальное хранилище

### Проблема: Push не приходят
**Решение:** 
- Проверьте разрешения браузера
- Убедитесь что Service Worker зарегистрирован
- Проверьте VAPID ключи в переменных окружения

### Проблема: Стили не применяются на карточках
**Решение:**
- JSON от редактора нужно применить в компоненте ProductCard
- Используйте экспортированный стиль в ваших компонентах
